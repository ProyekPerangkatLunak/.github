## Project Description ğŸ“•

The IoT system for providing melon nutrition based on PPM values â€‹â€‹is a system that uses PPM sensors to measure nutrient concentrations in irrigation water. The measurement results data is then sent to the server via the Wi-Fi module. The server will process the data and determine whether fertilizer or water needs to be applied. If necessary, the server will send commands to Arduino to activate the fertilizer or water pump.

This system has several advantages, including:
* Can provide proper nutrition according to plant needs.
* Can save fertilizer and water costs.
* Can reduce the risk of plant damage due to excessive fertilizer or water application.
